# Events

One file per booked event lives in this directory, named `[YYYY-MM-DD]-[event-or-program-name].md`.

Per `workflows/daily.md` Step 3, every session scans this directory for any event within 7 days and checks:
- Pre-event logistics email sent?
- Creator crew assigned?
- QR code and consent forms ready?
- Post-event: footage logged? Assignments sent? Families notified?
